Style attribute

The syntax of style attribute is :

< tagname style="property : value;" >

Uses of style attribute :

style attribute is used to change text colour. This is an aqua colour text.

It is also used to change the font-family. This is an "Times New Roman" font text.

It is used to adjust the font size. This is an 50px text

It is also used to background colour. The background of this colour is "bisque"

It is used to adjust the text alignment like center,right,left
