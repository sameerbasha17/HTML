THE PREFORMATED TAG

The preformated tag - < pre > , is used to display the content without ignoring the whitespaces and new lines in an element.

The difference between < p > tag and < pre > tag :

Rhyme using < p > tag:

Twinkle, twinkle, little star, How we wander what you are. Up above the world so high, Like a diamond in the sky.

Rhyme using < pre > tag: 
                Twinkle, twinkle, little star,
                How we wander what you are.
                Up above the world so high,
                Like a diamond in the sky.
